# 高考语文名句名篇默写数据库 (Gaokao Chinese Recitation Database)

这是一个关于高考语文“名句名篇默写”真题的结构化数据库，涵盖了从 2016 年到 2025 年的 115 道真题。本项目旨在通过精密的数据整理与深度分析，揭示高考命题的规律与趋势，为备考及研究提供参考。

## 🌟 项目亮点

- **数据详尽**：收录 115 道真题，覆盖全国各地主要试卷版本。
- **结构精密**：将真题精准对接部编版教材中的 74 篇背诵篇目。
- **深度分析**：基于数据统计出的高频考点分析及命题趋势解读。
- **易读输出**：提供按年份、按篇目分类的高质量文档。

## 📂 仓库结构

- **`高考考查.csv`**：核心数据库，包含题目、答案、年份、考区、对应篇目 ID 等多维数据。
- **`背诵篇目.csv`**：官方 74 篇背诵篇目的元数据（名称、对应课本、备注等）。
- **`官方课本.csv`**：部编版教材模块（必修、选择性必修）的映射表。
- **`docs/`**：
  - [📋 按年份题目汇编](docs/questions_by_year.md)：快速查阅历年分布。
  - [📚 按篇目题目汇编](docs/questions_by_piece.md)：针对特定课文的专项突破（答案已嵌入）。
  - [📊 深度分析报告](docs/README.md)：核心考点排行榜及命题趋势预测。

## 🏛️ 数据概览 (2016-2025)

通过对数据的智能审计，我们发现：

- **核心常青藤**：以《论语》、《琵琶行》、《赤壁赋》为首的篇目考查频次最高。
- **变革风向标**：试题已由简单的“机械记忆”全面转向“情境化运用”，对学生的情境带入与文本理解能力提出了更高要求。
